## Part 1: Basic Structure & Text

1. Create a new HTML file named `index.html`.  
2. Set up a basic HTML document structure with `<!DOCTYPE html>`, `<html>`, `<head>`, and `<body>`.  
3. Inside the `<body>`, add:
   - A main heading for the page title (e.g., your name or a topic you like).  
   - A subheading that describes what the page is about.  
   - A short paragraph introducing yourself or describing the topic.  
   - Add another paragraph that includes **bold**, *italic*, and **underlined** text.

## Part 2: Lists

6. Create a **nested list** (a list inside another list) ‚Äî for example, list different countries and their major cities.

## Part 3: Links and Images

7. Add a link to your favorite website ‚Äî make it open in a **new tab**.  
8. Add another link that points to a **different page** on your website (for example, `about.html`).  
9. Insert an image (can be from the web or your computer) with **alt text** describing it.

## Part 4: Tables

10. Create a table that displays information such as:
    - A list of students and their grades, **or**
    - A list of products and their prices.  
11. Your table should include:
    - A **table heading** row.  
    - At least **3 rows** of data.  
    - **Borders** and proper alignment of text.  
12. Add a **caption** above the table to describe what it represents.

---

## Part 5: Forms

13. Create a form for collecting user information.

    - A **submit** button.  
14. Add a heading above the form that says **"Registration Form"** (or similar).
